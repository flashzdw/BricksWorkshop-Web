# “全生命周期 机器人教育体系” 探索详情补充计划

## 1. 现状分析
当前网站的 `Curriculum.tsx` 组件中展示了“全生命周期 机器人教育体系”的三个阶段（VEX 123, VEX GO, VEX IQ）。每个阶段的卡片底部有一个“探索详情”的按钮区域，但目前仅为静态展示，没有实际的交互与详细内容支撑。

## 2. 拟议更改 (Proposed Changes)
通过查找 VEX 官方及相关教育资料，我们已经整理了 VEX 123、VEX GO 和 VEX IQ 的核心特点与培养目标。计划在 `Curriculum.tsx` 中引入弹窗（Modal）交互，当用户点击“探索详情”时，以优雅的动画展示该体系的详细内容。

### 步骤 1：扩充课程数据结构
在 `Curriculum.tsx` 的 `courses` 数组中，为每个课程对象补充更详细的数据字段（如 `features`, `outcomes`, `techSpecs` 等），具体内容如下：

*   **VEX 123 (4-7岁)**
    *   **核心特点**：触碰式编程（Touch & Go）、卡片编程（无屏编程）、图形化编程（VEXcode 123）。
    *   **培养目标**：激发科技好奇心，建立基础逻辑与序列概念，培养早期的计算思维。
    *   **硬件特色**：互动性高、发声发光的智能机器人底盘，配有独立编码器。

*   **VEX GO (8-10岁)**
    *   **核心特点**：模块化彩色拼插结构（易于搭建与整理）、图形化积木式编程（基于 Scratch）。
    *   **培养目标**：培养空间想象力和问题解决能力，完成从具象操作到抽象逻辑的过渡，建立对工程的初步感知。
    *   **硬件特色**：内置陀螺仪的主控器、智能电机、电磁铁与辨色仪等丰富外接件。

*   **VEX IQ (11-14岁)**
    *   **核心特点**：高级工程结构设计、支持多种编程语言（块编程、C++、Python）、直通全球规模最大的机器人赛事。
    *   **培养目标**：锻炼复杂问题解决能力、团队协作精神与赛事抗压能力，为国内外升学及科技特长生打下坚实基础。
    *   **硬件特色**：多功能主控器、视觉传感器、第二代测距仪、高精度智能电机。

### 步骤 2：引入交互状态与弹窗组件
1.  引入 `useState` 和 `AnimatePresence` (来自 `framer-motion`)。
2.  在 `Curriculum.tsx` 中增加一个 `selectedCourse` 状态，用于记录当前被点击查看详情的课程。
3.  实现一个全屏遮罩及弹窗组件（Modal），在用户点击“探索详情”时弹出，显示上述补充的图文与特性列表。
4.  弹窗应包含关闭按钮，并且支持点击遮罩层关闭，具有平滑的淡入淡出动画。

### 步骤 3：优化卡片点击事件
修改原有卡片中“探索详情”区域的结构，使其不仅是静态的 `span`，而是可点击的交互元素，或者让整个卡片具有可点击的属性并绑定 `onClick` 事件，触发弹窗展示。

## 3. 假设与决策
*   **展现形式**：使用单页面内的 Modal 弹窗比跳转新页面拥有更好的连贯性和用户体验，也符合当前单页滚动落地页的设计风格。
*   **样式库**：使用项目中已有的 `framer-motion` 处理动画，使用 `lucide-react` 提供弹窗内的图标（如关闭图标、列表勾选图标等）。
*   **文案**：资料基于检索到的 VEX 官方及常见教育资料生成，贴合“全生命周期”这一主题。

## 4. 验证步骤
1.  运行 `npm run dev` 启动项目。
2.  滚动至“全生命周期 机器人教育体系”部分。
3.  点击 VEX 123、VEX GO、VEX IQ 的卡片或“探索详情”按钮。
4.  确认带有详细信息和精美排版的弹窗顺利滑出，内容准确无误。
5.  点击弹窗的关闭按钮或背景遮罩，确认弹窗能平滑关闭。
