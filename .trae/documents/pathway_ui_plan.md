# “从零基础到世界舞台”部分 UI 更改计划

## 摘要
本计划旨在修改首页“从零基础到世界舞台”（赛事路径）部分步骤图标的视觉样式。根据用户的反馈，将原有的**较小正圆形**图标修改为**更大的圆角正方形（Squircle）**，以增强视觉冲击力和现代感。

## 当前状态分析
- **文件位置**：`src/components/Pathway.tsx`
- **当前形状**：使用 `rounded-full` 实现的正圆形。
- **当前尺寸**：外层容器为 `w-20 h-20`（80x80px），内部 Lucide 图标为 `w-8 h-8`（32x32px）。
- **连接线位置**：连接步骤的横线通过 `top-10` 绝对定位，正好位于 80px 高度的中心。
- **背景光晕**：背景渐变光晕也使用了 `rounded-full` 形状。

## 提议的更改
需要修改 `src/components/Pathway.tsx` 文件，具体步骤如下：

1. **调整外层容器的尺寸与形状**：
   - 将 `w-20 h-20` 修改为 `w-28 h-28`（112x112px），使图标更加醒目。
   - 将 `rounded-full` 修改为 `rounded-3xl`（24px 圆角），形成平滑的圆角正方形。
2. **调整内部图标尺寸**：
   - 由于外框变大，将内部的 Lucide 图标尺寸从 `w-8 h-8` 相应放大至 `w-12 h-12`。
3. **调整背景光晕（Glow）形状**：
   - 将光晕层的 `rounded-full` 修改为 `rounded-3xl`，使其与新容器形状保持一致。
4. **修正连接线的垂直居中位置**：
   - 容器高度变为 `h-28`（112px）后，其垂直中心点变为 56px。因此需要将连接线的绝对定位从 `top-10`（40px）修改为 `top-14`（56px），以保证横线依然精准穿过各个圆角正方形的中心。

## 假设与决策
- **设计决策**：选择 `w-28` 和 `rounded-3xl` 是为了在“明显变大”与“整体排版协调”之间取得平衡，既提升了显眼程度，又不会挤占过多的横向空间。
- **交互不变**：保留了原有的 `group-hover:scale-110` 悬浮放大动画和边框阴影效果，确保原有的良好交互体验不丢失。

## 验证步骤
1. 启动本地开发服务器。
2. 在浏览器中访问首页，滚动至“从零基础到世界舞台”区域。
3. 检查 4 个步骤图标是否已变为较大的圆角正方形。
4. 确认各步骤之间的水平连接线是否完美穿过每个图标的中心。
5. 将鼠标悬浮在图标上，确保原有的放大动画与光晕增强效果正常运行，且图标内部元素不出现错位。
