# 网站文案真实化优化计划 (Copywriting Optimization Plan)

## 1. Summary (摘要)
基于 VEX Academy 官方博客（西北地区首家 VEX 学苑）与小红书主页（卓越科技BricksWorkshop）提取的真实信息，优化当前网站的文案。将原本较为通用、模板化的描述替换为包含真实历史、核心教育理念（“不凑合”）、具体战队（8009系列）以及实际参与赛事（NASA太空挑战、WRC等）的生动文案。提升网站的真实感与品牌温度。

## 2. 整理出的可用真实信息 (Current State Analysis)
从提供的两个信息源中，提取出以下可用于替换网站通用文案的“真实细节”：
1. **品牌与历史**：机构名为“卓越科技 BricksWorkshop”，成立于 2015 年 7 月 11 日。2018 年加入 VEX 课堂，2022 年 10 月正式升级为西北地区首家 VEX 学苑官方校区。
2. **核心教育理念**：
   - Slogan：**“关于孩子，我们不想凑合”**。教育要有初心、情怀，跟随前沿。
   - 教学特色：教学是“倾听与观察”。教师不仅是传授者，更是倾听者与观察者，接收学生、家长与赛场的真实反馈。
   - 价值观：“以学生为中心”。
3. **校区位置与环境**：坐落于汉中市汉台区滨江西路天汉长街 E 区 – E1。拥有大面积玻璃门窗、原木色机器陈列柜、简洁宽敞的书桌以及大面积的练习场地，环境通透。
4. **课程特色**：涵盖 VEX 123、VEX GO、VEX IQ 等。强调“建模是专注度非常高的游戏”，并积极推动 VEX 课程进“校园”。
5. **战队与赛事**：
   - **战队**：拥有“8009”系列战队（包含 8009A, 8009S“铜车马”, 8009C, 8009W 等）。
   - **赛事记录**：不仅参与 VEX 世锦赛，还参加过 NASA 全球太空挑战、南工杯、WRC 世界机器人大会、华科大邀请赛、科协比赛等。

## 3. Proposed Changes (拟修改方案)

### 3.1 修改 `src/components/Hero.tsx`
- **What**：优化主视觉区域的副标题文案。
- **Why**：原副标题“专注力与创造力的起点”较为常规。
- **How**：融入其最核心的 Slogan：“关于孩子，我们不想凑合”。例如修改为：“西北地区首家 VEX 学苑，一家‘不凑合’的校区。关于孩子，我们从不妥协，探索未来科技的无限可能。”

### 3.2 修改 `src/components/About.tsx`
- **What**：重写品牌介绍和团队理念，加入真实发展历程。
- **Why**：原介绍缺乏时间沉淀的厚重感与具体的教育情怀。
- **How**：
  - 在介绍段落明确写出“成立于 2015 年，2022 年升级为西北首家官方校区”。
  - 将“顶尖师资团队”的描述修改为：“我们的教师是倾听者与观察者。我们接收来自学生、家长及赛场的真实反馈，以学生为中心，因材施教。”
  - 将“专业校区环境”的描述具体化为：“坐落于天汉长街 E 区，大面积玻璃门窗与原木色陈列柜……”

### 3.3 修改 `src/components/Campus.tsx`
- **What**：使用具体的地址与真实的环境描述。
- **Why**：原描述“沉浸式的创造空间”不够具象。
- **How**：修改文案为：“坐落于汉中市滨江西路天汉长街。自然光透过大面积玻璃门窗，原木色机器陈列柜、宽敞的书桌与大面积练习场地一览无余，通透的环境让学习更加轻松愉悦。”

### 3.4 修改 `src/components/Curriculum.tsx`
- **What**：在课程描述中融入真实教学场景细节。
- **Why**：让课程介绍更贴近其日常真实的教学状态。
- **How**：
  - 在 VEX 123 课程中，提及“小课堂”中的实际应用。
  - 在进阶课程中，突出小红书提到的“建模是专注度非常高的游戏”这一观点。

### 3.5 修改 `src/components/Pathway.tsx`
- **What**：加入真实的赛事名称与战队名称。
- **Why**：原赛事路径只有泛泛的“省市赛、世锦赛”，缺乏该机构独特的战绩背书。
- **How**：
  - 在“校队与省市赛”中点名加入真实的“8009 战队（如 8009A、8009S 铜车马）”。
  - 在比赛列举中加入实际参加的“NASA 全球太空挑战”、“WRC 世界机器人大会”、“华科大邀请赛”、“南工杯”等，提升真实感和专业度。

## 4. Assumptions & Decisions (假设与决策)
- **假设**：小红书上的 8009 战队和各类赛事活动代表了校区当前的最高实力与活跃度，因此将其作为品牌实力背书展示在网站上是合适的。
- **决策**：将核心的“不凑合” Slogan 作为贯穿全站的情感主线，这能很好地建立家长对机构的信任感。

## 5. Verification steps (验证步骤)
1. 检查 `npm run build` 或 `npm run check` 是否能正常通过，确保没有破坏 React 组件结构。
2. 启动 `npm run dev` 预览网页，检查文案替换后在不同屏幕尺寸下的排版和可读性。
3. 确认所有新加入的真实细节（年份、战队号、赛事名、地址）拼写无误。
