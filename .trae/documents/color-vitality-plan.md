# UI 色彩活力提升计划

## 摘要 (Summary)
为了打破当前网站全黑白灰（Zinc）的沉闷感，提升作为教育机构网站的活力与吸引力，本计划将为页面的关键元素（数据统计百分比、VEX 各教育体系名称等）引入明亮的强调色。核心策略是使用科技蓝（Blue）作为全局数据强调色，并为 VEX 的三个不同体系（123、GO、IQ）分别分配专属的品牌强调色。

## 现状分析 (Current State Analysis)
- **数据统计 (`Statistics.tsx`)**：当前的数字与百分比均为深灰色 (`text-zinc-900`)，虽然高级但略显平淡。
- **课程体系 (`Curriculum.tsx`)**：VEX 123、VEX GO、VEX IQ 等体系名称均为深灰色，缺乏辨识度和活泼感。
- **赛事路径 (`Pathway.tsx`)**：阶段标题为纯深灰。
- **整体反馈**：当前的 Apple 极简风格保留得很好，但对于儿童/青少年教育网站，缺少能够抓住眼球的“高光色彩”。

## 具体修改方案 (Proposed Changes)

### 1. `src/components/Statistics.tsx` - 数据强调色
- **What**: 将所有大号数字和后缀（如 `+`、`%`）修改为科技蓝 (`text-blue-600`)。
- **Why**: 蓝色传递出专业、科技和信任感，同时在纯白背景上具有极佳的对比度。
- **How**: 修改 `text-zinc-900` 为 `text-blue-600`。

### 2. `src/components/Curriculum.tsx` - VEX 体系专属强调色
- **What**: 为 VEX 123、VEX GO、VEX IQ 三个课程名称分别设定专属颜色：
  - VEX 123：红色 (`text-red-500`)，代表启蒙与热情。
  - VEX GO：绿色 (`text-emerald-500`)，代表探索与成长。
  - VEX IQ：蓝色 (`text-blue-600`)，代表科技与进阶。
- **Why**: 为不同阶段的课程赋予独立的色彩心智模型，不仅增加了页面的丰富度，也让家长更容易区分不同的课程体系。
- **How**: 
  - 在 `courses` 数组的每一项中新增一个 `colorClass` 字段。
  - 渲染列表和模态框时，将课程名称 (`<h3>`) 应用对应的 `colorClass`。

### 3. `src/components/Pathway.tsx` - 赛事路径节点强调
- **What**: 将赛事路径中的小圆点和标题中的关键名词（如“零基础”、“省市赛”、“世锦赛”）应用科技蓝 (`text-blue-600` 或 `bg-blue-600`)。
- **Why**: 引导用户视线，让进阶路径显得更加清晰和令人振奋。
- **How**:
  - 将黑色小圆点 `bg-zinc-900` 改为 `bg-blue-600`。
  - 将各阶段标题 `<h3>` 改为 `text-blue-600`，或者整体保持深灰但保留节点的高亮。

## 假设与决策 (Assumptions & Decisions)
- **决策**：保留已修改好的大标题彩虹渐变色（“红黄蓝”渐变），因为它们能统领全站的活泼基调。
- **决策**：局部的强调色（如数字、VEX 体系名）采用纯色而非渐变色，以避免页面色彩过于凌乱，做到“大处有彩虹，局部有专色”的平衡。

## 验证步骤 (Verification steps)
1. 运行 `npm run dev` 启动项目。
2. 检查 `Statistics` 组件，确认数字和百分比显示为亮蓝色。
3. 检查 `Curriculum` 组件，确认 VEX 123 显示为红色，VEX GO 为绿色，VEX IQ 为蓝色（包括列表和弹出的详情页）。
4. 检查 `Pathway` 组件，确认时间轴的节点圆点已变为蓝色。
