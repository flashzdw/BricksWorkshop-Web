# 卓越科技BricksWorkshop 内容扩充与真实图片替换计划

## 1. 目标与现状分析
- **目标**：大幅增加网页的文字与板块内容，使其看起来更丰满、专业；将目前使用的通用 Unsplash 占位图替换为真实的 VEX 机器人（VEX 123/GO/IQ）相关图片及真实赛场/课堂图片。
- **现状**：
  - 网站目前包含 Hero、Statistics、About、Curriculum、Gallery 和 Footer。
  - 各个板块的文字描述较短，缺乏深度的教育理念、常见问题等支撑内容。
  - Curriculum 和 Gallery 使用的是 Unsplash 上的通用高清图，与 VEX 机器人的强关联性不足。

## 2. 拟实施的变更

### 2.1 替换为真实 VEX 图片
- **涉及文件**：`src/components/Curriculum.tsx`, `src/components/Gallery.tsx`
- **操作**：
  - **课程体系 (Curriculum)**：使用网络开源的真实 VEX 产品图片替换现有占位图。
    - VEX 123 替换为真实的 VEX 123 机器人图片。
    - VEX GO 替换为真实的 VEX GO 搭建图片。
    - VEX IQ 替换为真实的 VEX IQ 机器人图片。
  - **风采展示 (Gallery)**：将 5 张通用图片替换为开源的 VEX 真实课堂、比赛现场的实拍照片，以最大程度还原小红书上的真实感。

### 2.2 扩充“关于我们”与教育理念 (About)
- **涉及文件**：`src/components/About.tsx`
- **操作**：
  - 在原有的三大特点（极致专注、西北首家、寓教于乐）基础上，增加一段“教育理念与愿景”的深度文案。
  - 增加关于“汉中滨江路校区”的硬件设施、师资背景的文字描述，提升机构的专业信任度。

### 2.3 新增“常见问题”板块 (FAQ)
- **涉及文件**：`src/components/FAQ.tsx`（新建）, `src/pages/Home.tsx`, `src/components/Navbar.tsx`
- **操作**：
  - 新建一个手风琴（Accordion）样式的 FAQ 板块，解答家长最关心的几个问题：
    1. 孩子零基础可以报名学习吗？
    2. VEX 机器人课程和其他乐高课程有什么区别？
    3. 学习 VEX 对孩子未来的升学或科技特长生有帮助吗？
    4. 滨江路校区的上课频率和班型是怎样的？
  - 在首页组件中引入该板块，并在 Navbar 中增加对应的锚点导航。

### 2.4 新增“学习路径与赛事”板块 (Pathway)
- **涉及文件**：`src/components/Pathway.tsx`（新建）, `src/pages/Home.tsx`
- **操作**：
  - 增加一个展示孩子从“零基础启蒙”到“参与国际 VEX 机器人亚洲公开赛/世锦赛”的进阶路径图。
  - 配合文字说明机构在赛事辅导方面的专业能力。

## 3. 假设与决策
- **图片资源来源**：由于无法直接通过脚本批量抓取特定小红书账号的受保护图片，我将使用 GitHub 上公开的 STEM/VEX 教育开源项目（如 RPAIR 团队的 VEX 真实赛场和产品图）作为高质量的替代。这些图片完全是 VEX 123/GO/IQ 的真实照片，完美契合要求。
- **内容编写**：FAQ 和学习路径的文案将根据 VEX 机器人教育的行业标准和“西北首家VEX学苑”的定位进行专业化拟写。
- **UI 风格**：新增的 FAQ 和 Pathway 板块将继续严格遵循现有的极简白底+彩虹点缀风格，确保动画连贯性。

## 4. 验证步骤
1. 检查 Curriculum 版块，确认图片已变为真实的 VEX 123/GO/IQ 机器人。
2. 检查 Gallery 版块，确认图片已替换为 VEX 相关的课堂/比赛真实场景。
3. 滚动浏览页面，确认 About 版块内容已明显增多。
4. 确认新增了 FAQ 和 Pathway 板块，点击手风琴展开动画正常。
5. 检查导航栏是否已同步更新并能正确跳转到新增板块。
